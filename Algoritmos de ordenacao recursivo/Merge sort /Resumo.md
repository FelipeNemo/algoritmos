# Merge sort 

## Descrição
Merge sort é um algoritmo de divisão e conquita em estruturas lineares

## Complexidade
n logn

n = constante
É explicado na conquista da estrutura.
log n = curva
Feito na divisão da estrutura

## Caracteristicas:
- ALgoritmo mais rápido(divisão de um problema em partes menores)
- Algoritmo recursivo
- Estabelece um pivot
- Divide o vetor em duas partes até acabar
- N Log n
N passadas(Colocar o pivot) * n Logarimica(Divisão do vetor)
